AxisColor Picker


This tool allows you to easily select and customize colors and axis of your graph for your Computer Graphics OpenGL projects. You can select and copy coordinates and color easily and apply it on your code with this tool. This tool allows you to copy:

  • Color Codes: rgb0-1(0.000, 0.000, 0.000)
  • Coordinate Points: glVertex2i(543, 700);

on your project code.
